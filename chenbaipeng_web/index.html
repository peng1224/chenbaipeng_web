医美实习

profit first information gap distorted values disillusion

2023年夏季，我进入医美行业。在那里，我观察到设计与心理学有时被误用为诱导消费的工具，而非提升体验的助力。

这段经历让我决定离开。我意识到，设计不该只是粉饰利润的滤镜，更不应建立在信息差的博弈之上。

我开始渴望一种诚实的逻辑：在那里，因果清晰可见，价值不需要被包装，而是一次次被精准地验证。我决定去寻找那个更纯粹的世界。
